Common Issues and Solutions

  • Connection String Missing: Ensure AZURE_STORAGE_CONNECTION_STRING environment variable is set
  • Invalid Format: Check that the connection string contains DefaultEndpointsProtocol, AccountName, and AccountKey
  • URL Encoding: Connection strings from environment variables might be URL-encoded
  • Container Access: Verify that the storage account allows container creation and blob operations
  • Network Issues: Check firewall rules and network connectivity to Azure Storage
